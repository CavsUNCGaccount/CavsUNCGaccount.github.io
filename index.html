(Important) For hw5:

1. Generate your own GitHub API key/token.
2. Create a .env file to store the key In your project directory, create a new file called .env.
3. Paste this line to the .env file with your API key replacing the text API key here: GITHUB_TOKEN=API key here
4. Make sure to install Node.js and npm as the server depends on these.
5. Install express, dotenv, and axios using the terminal commands:
npm install express
npm install dotenv
npm install axios
